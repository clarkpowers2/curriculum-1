# Private Tutor

A lightweight, browser-based private tutoring application. No build step, no server required — just open `index.html` in any modern browser.

## Features

| Feature | Description |
|---|---|
| **Lessons** | Create and manage rich text lessons organised by subject |
| **Flashcards** | Flip-card study tool with subject filtering |
| **Quiz** | Multiple-choice quizzes auto-generated from your flashcards |
| **Progress** | Stats overview and quiz history |

All data is stored in `localStorage`, so your lessons and progress persist between sessions.

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Explore the sample lessons and flashcards, or add your own.

## File Structure

```
index.html   – Application shell and HTML markup
style.css    – All styles (CSS custom properties, responsive layout)
app.js       – Application logic (no external dependencies)
```
